# AI Resume Screening System

## 🚀 Overview
This project uses machine learning and NLP to automatically screen resumes and match them with job descriptions.

##  Features
- Resume parsing
- Skill extraction
- Candidate ranking
- Job matching

##  Tech Stack
- Python
- Scikit-learn / TensorFlow
- NLP (spaCy / NLTK)

##  How to Run
1. Clone repo
2. Install requirements:
   pip install -r requirements.txt
3. Run:
   python resume-screening.py

## Future Improvements
- Web interface
- Deep learning models
- Real-time resume upload
